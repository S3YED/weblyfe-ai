# Source Instructions for Weblyfe.ai Case Studies

## Primary Source: weblyfe.nl

Use https://weblyfe.nl/projects as the main source for:
- Project names
- Project descriptions
- Live website URLs
- Project images (from CDN)

## Key Projects for AI Case Studies

### 1. Booth.me (Boooth)
- **Main site:** https://booth.me or search weblyfe.nl for "Boooth"
- **Booking system:** https://booking.booth.me

### 2. Titan Transfers
- **Booking:** https://book.titantransfers.be

### 3. LPS Pilates
- **Site:** https://lpspilates.nl (or similar)
- **AI angle:** SimplyBook integration, AI-powered scheduling

### 4. Viva Fashion

- **AI angle:** AI content generation for products

## How to Extract Project Info

1. Visit weblyfe.nl/projects
2. Each project card has:
   - Name
   - Description
   - Live URL
   - Screenshot/image

3. For screenshots, use the CDN URLs from the page:
   - `https://cdn.prod.website-files.com/615bea49d9d3ed2611fdb246/...`
