# Boooth — AI-Powered Photo Marketing Platform

## The Client
**Name:** Jason Tuhumena  
**Company:** Boooth  
**Industry:** Event Technology & Corporate Branding  
**Website:** https://boooth.nl / https://booking.booth.me  
**Experience:** 11+ years in photo marketing for corporate events

## The Challenge
Boooth needed to position their service beyond simple "booth rentals" and establish themselves as a strategic corporate branding tool. They required:
- A booking system that showcases their premium positioning
- AI-powered configurator for event packages
- Seamless integration between marketing site and booking flow
- Professional presence that reflects 11+ years of expertise

### Pain Points
- Manual booking process created friction
- Difficult to communicate value proposition beyond "photo booth"
- No way to showcase portfolio of corporate clients
- Needed to differentiate from commodity booth rentals

## The Solution
Weblyfe built a complete AI-powered booking ecosystem that transforms how corporate clients engage with Boooth's services.

### Technologies Used
- **Platform:** Webflow (marketing site)
- **Booking System:** Custom booking configurator at booking.booth.me
- **AI Features:** Smart package recommendations based on event type
- **Payment:** Integrated payment processing
- **Automation:** Automated confirmations and follow-ups
- **Brand Design:** Complete visual identity and positioning

### Key Features

#### 1. **AI Booking Configurator**
Smart configurator that recommends optimal packages based on:
- Event type (corporate party, product launch, team building)
- Guest count
- Venue size
- Budget range
- Brand activation goals

#### 2. **Strategic Positioning**
Repositioned from "photo booth rental" to "strategic tool for corporate branding and unforgettable guest experiences"

#### 3. **Portfolio Showcase**
Dedicated client showcase highlighting 11+ years of corporate events

#### 4. **Seamless Booking Flow**
- Event details → Package selection → Customization → Checkout
- Real-time availability checking
- Instant confirmation
- Automated pre-event communications
