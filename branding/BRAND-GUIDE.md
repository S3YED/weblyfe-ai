## 🎨 Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| **Primary Dark** | `#062421` | Full-bleed backgrounds, panels |
| **Primary Text** | `#E9F6F1` | Headlines, body text on dark backgrounds |
| **Secondary Text** | `#A5C2BC` | Captions, secondary paragraphs |
| **Accent (Peach)** | `#F7D69E` | Highlights, small brand marks, icons |
| **Divider/Rule** | `#9FBEB6` | Subtle separators, rules |

### CSS Variables
```css
:root {
  --bg-primary: #062421;
  --text-primary: #E9F6F1;
  --text-secondary: #A5C2BC;
  --accent: #F7D69E;
  --divider: #9FBEB6;
}
```

## 🔤 Typography

**Primary Font:** Rethink Sans

| Level | Weight | Size | Line Height | Usage |
|-------|--------|------|-------------|-------|
| **Display** | Bold | 72-120px | 1.02-1.15 | Hero headlines |
| **H1** | Bold | 36-56px | 1.05-1.2 | Primary headlines |
| **H2** | Semi-Bold | 24-32px | 1.1-1.3 | Section headings |
| **H3** | Medium/Semi-Bold | 18-22px | 1.2-1.4 | Subheads |
| **Body** | Medium | 16-18px | 1.45-1.6 | Paragraphs |
| **Caption** | Medium | 12-14px | 1.3-1.5 | UI text |

### Font Import
```css
@import url('https://fonts.googleapis.com/css2?family=Rethink+Sans:wght@400;500;600;700&display=swap');
```

## 🏷️ Logo

- **Location:** `/branding/Logo.png`, `/branding/Brandmark.png`
- **Style:** Geometric, rounded sans-serif with arrow in the "e"
- **Tagline:** All caps, thin weight, wide tracking

---

## ✨ Brand Vibe

- **Professional** — Dark green signals reliability and quality
- **Modern** — Clean, minimal, digitally focused
- **Forward-looking** — Arrow in logo suggests motion/direction
- **Friendly** — Rounded forms, pale backgrounds, warm peach accent
- **Trustworthy** — Crafted, deliberate, refined

---

## ⚠️ Do NOT Use

- Purple/Indigo (#6366f1) ❌
- Generic bright colors ❌
- White backgrounds for main sections ❌ (use dark primary)
- Robotic/cold tones ❌
